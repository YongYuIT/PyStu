# 数据输入

## 数据预处理（DataPreprocessing）

1. 图片标准化：将图片扩展为1:1，不够的部分补上黑色
2. 图片大小一致：将图片缩放为100*100像素
3. 图片黑白化：将彩色图片转化为黑白图片
4. 图像增广：
   * 将图片左右颠倒
   * 将图片亮度进行1/2和2倍缩放
