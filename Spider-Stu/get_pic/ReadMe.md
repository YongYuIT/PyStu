# 安装 splash

## ubuntu更新国内源

图形界面操作方法：

设置(Settings)-->关于(About)-->软件和更新(Sofeware Updates)-->Ubuntu软件(Ubuntu software)
下载自(Download from)

默认是：Server for United States
改成：中国，mirrors.aliyun.com

执行

~~~shell
sudo apt-get update
sudo apt-get upgrade
~~~
